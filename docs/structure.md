# Structure du projet crypto_poly_bet

- src/ : code source principal
- contracts/ : smart contracts Ethereum
- backend/ : API, logique serveur
- frontend/ : interface web
- mobile-demo/ : démo mobile (smartphone)
- tests/ : tests unitaires et d'intégration
- docs/ : documentation technique et fonctionnelle

Cette structure est adaptée à une MVP orientée prediction market, avec une démo mobile.
