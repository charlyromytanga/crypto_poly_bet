# Roadmap Crypto Voting — Prediction Game

## Vision
Transformer la prédiction du futur en expérience sociale, fun et accessible à tous.

## Phases

### Phase 1 : MVP + Marchés basiques
- Landing page attractive
- Marchés variés (sport, politique, pop culture, tech)
- Paris simples (Oui/Non)
- Smart contracts automatisés
- UX simplifiée (onboarding rapide)

### Phase 2 : Social features + Gamification
- Profils publics
- Leaderboard (top predictors)
- Badges, niveaux, streaks
- Partage social natif (X, Discord)
- Suivi d’utilisateurs

### Phase 3 : Mobile + Scaling
- Tunnel HTTPS pour démo mobile (ngrok + QR code)
- UI mobile-friendly (swipe, Yes/No)
- Group betting (pool entre amis, ligue privée)
- Live betting (matchs, événements)

### Phase 4 : Token + Économie interne
- Token natif (optionnel)
- Incentives (bonus, free bets)
- Réduction de fees
- Gouvernance communautaire

## Différenciateurs
- Gamification hardcore
- Social + ego (profils, leaderboard, partage)
- Marchés funs (pop culture, trends)
- Résolution ultra fiable (oracles, vote, API)
- UX simplifiée (wallet abstrait, onboarding)

## Points techniques clés
- Smart contracts Ethereum / L2
- Oracles pour résolution des marchés
- Wallet integration simplifiée
- Sécurité (audit, tests, gestion des fonds)
- Régulation (KYC, AML, restrictions)

## À garder en tête
- Moins "outil financier", plus "jeu social addictif"
- UX simple + dopamine loop
- Adoption mainstream
- Compliance dès le début
