# Crypto Voting

Projet de vote et prédiction crypto, architecture microservices.

## Stack technique

- **Frontend** :
  - React (SPA, Vite)
  - CSS custom, responsive, mobile-first
  - Routing via react-router-dom
  - Docker (nginx pour la prod)

- **Backend** :
  - Node.js (API REST)
  - Express
  - Docker

- **Smart Contracts** :
  - Solidity (Ethereum)
  - Oracles pour la résolution
  - Interaction via web3.js

- **Démo mobile** :
  - Accès via tunnel HTTPS (ngrok)
  - QR code pour onboarding instantané

- **DevOps & Sécurité** :
  - Docker-compose pour orchestration
  - Nginx pour reverse proxy
  - Rapport sécurité dédié

- **Autres** :
  - Scripts shell pour setup et tests
  - Documentation et guides onboarding

---

La stack est conçue pour la scalabilité, la rapidité de déploiement, et la sécurité.
