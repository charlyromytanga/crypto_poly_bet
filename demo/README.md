# Demo Crypto Voting — Prediction Game

Cette démo permet de présenter le projet Crypto Voting selon la roadmap : prédiction sociale, UX simplifiée, tunnel HTTPS mobile, QR code, landing attractive.

## Fonctionnalités de la démo
- Landing page attractive (nginx)
- Tunnel HTTPS (ngrok) pour accès mobile
- QR code pour accès rapide
- Proxy vers API et smart contracts
- Checklist technique pour la démo

## Utilisation
1. Démarrer les conteneurs Docker (landing, API, smart contracts, oracle)
2. Lancer ngrok pour exposer le frontend en HTTPS
3. Générer le QR code pour accès mobile
4. Tester la landing et les marchés de prédiction

## Points techniques
- Marchés funs (sport, pop culture, tech)
- Paris Oui/Non
- Smart contracts automatisés
- UX onboarding rapide
- Sécurité et monitoring
